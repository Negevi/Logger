Change logger settings on the respective .json file
For better organization, it is recommended that you create a "Logs" folder, use "./Logs/" as path and json set to true.
Default datefmt: "%d-%m-%Y %H:%M:%S"

Available colors:
"red"
"green"
"yellow"
"blue"
"magenta"
"cyan"
